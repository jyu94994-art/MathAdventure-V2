# 数学大冒险 Math Adventure V2.0

这是一个带后端的网页游戏版本，包含：

- 用户注册登录
- Python 后端服务器
- SQLite 数据库
- 云端存档
- 排行榜接口
- 管理后台
- 原有本地离线试玩能力

## 本地启动

双击 `启动服务器.bat`。

启动后浏览器打开：

- 游戏入口：`http://127.0.0.1/`
- 管理后台：`http://127.0.0.1/admin.html`

## 默认管理员

- 用户名：`admin`
- 密码：`admin123`

正式给别人使用前，建议修改默认管理员密码。

## 数据库

首次启动时会自动创建：

`data/math_adventure.db`

里面保存用户、登录会话、云端存档和排行榜成绩。

## 让别人也能玩

如果只是在同一个局域网内试玩：

1. 你的电脑运行 `启动服务器.bat`
2. 查看你的局域网 IP，例如 `192.168.1.23`
3. 别人在浏览器打开 `http://你的IP/`

如果要公网访问，需要把这个项目部署到云服务器，或使用 Nginx 反向代理到正式域名。
