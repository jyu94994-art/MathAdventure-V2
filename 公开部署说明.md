# 数学大冒险公开部署说明

## 推荐方式：Render

Render 可以把这个 Python 网页游戏部署成公开网址，例如：

`https://math-adventure-v2.onrender.com`

## 已准备好的文件

项目里已经包含：

- `server.py`：后端服务器
- `requirements.txt`：Python 依赖文件
- `Procfile`：通用启动配置
- `render.yaml`：Render 部署配置
- `index.html`：游戏页面
- `admin.html`：管理后台
- `data/math_adventure.db`：SQLite 数据库

## 部署步骤

### 第一步：上传到 GitHub

1. 打开 GitHub
2. 新建一个仓库，例如 `MathAdventure-V2`
3. 把整个 `MathAdventure-V2` 文件夹里的所有文件上传进去

注意：不要只上传 `index.html`，必须上传整个项目。

### 第二步：连接 Render

1. 打开 `https://render.com`
2. 注册或登录账号
3. 点击 `New +`
4. 选择 `Web Service`
5. 连接你的 GitHub 仓库
6. 选择 `MathAdventure-V2`

### 第三步：填写配置

如果 Render 没有自动识别，可以手动填写：

- Environment：`Python`
- Build Command：`pip install -r requirements.txt`
- Start Command：`python server.py`

然后点击部署。

### 第四步：访问游戏

部署成功后，Render 会给你一个公开网址。

游戏入口：

`https://你的Render网址/`

管理后台：

`https://你的Render网址/admin.html`

## 默认管理员

- 用户名：`admin`
- 密码：`admin123`

正式公开前建议修改默认管理员密码。

## 关于数据库

当前版本使用 SQLite。

适合：

- 试玩
- 小范围测试
- 演示给别人看

如果你要长期正式运营，建议后续升级为：

- PostgreSQL
- MySQL
- 云数据库

否则免费平台重启或重新部署时，数据库可能会被重置。

## 如果只想局域网访问

不用部署到 Render。

直接在你的电脑双击 `启动服务器.bat`（右键以管理员身份运行），然后让同一个 Wi-Fi 下的人访问：

`http://你的电脑IP/`
